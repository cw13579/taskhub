 # TaskHub 渠道任务协作平台 V1.1
 
 ## 概述
 TaskHub 是一个面向渠道伙伴的任务下发与协作平台，支持管理员统一管理多公司任务，渠道伙伴按公司查看、处理任务。
 
 ## 程序启动入口文件
 
 | 入口 | 文件 | 说明 |
 |------|------|------|
 | 用户端（PC） | `index.html` | 渠道伙伴使用，输入名字+选择公司进入 |
 | 管理端 | `admin.html` | 管理员使用，统一管理所有公司任务 |
 | 移动端 | `mobile.html` | 手机浏览器自动跳转，功能与用户端一致 |
 
 启动方式：用任意 HTTP 服务器托管本目录即可，移动端访问时自动跳转 `mobile.html`。
 
 ## 数据存储路径
 
 所有数据存储在浏览器 `localStorage` 中，无需后端数据库：
 
 | Key | 内容 | 作用域 |
 |-----|------|--------|
 | `th3_global` | 任务、公司列表、管理员Cooper、审批、用户-公司映射 | 全局共享 |
 | `th3_users` | 用户个人Cooper链接、任务备注/打标 | 按用户隔离 |
 | `th3_current_user` | 当前登录用户名 | 当前浏览器 |
 | `th3_fd` | 是否强制桌面端（跳过移动端跳转） | 当前浏览器 |

 ## 依赖环境
 
 - 无需安装任何依赖，纯前端静态页面
 - 推荐浏览器：Chrome / Edge / Safari 最新版
 - 需要一个 HTTP 服务器来托管文件（直接双击打开 HTML 也可运行，但移动端跳转会失效）
 
 ## 运行启动命令
 
 ```bash
 # 方式1：Python 内置服务器
 cd release-v1.1
 python -m http.server 8090
 
 # 方式2：Node.js
 npx serve release-v1.1 -p 8090
 
 # 方式3：直接双击 index.html 打开（移动端跳转不可用）
 ```
 
 启动后访问：
 - 用户端：`http://localhost:8090/index.html`
 - 管理端：`http://localhost:8090/admin.html`
 - 移动端：手机浏览器访问同一地址，自动跳转
 
 ## 功能模块
 
 ### 用户端
 - 登录：首次输入名字 + 选择公司，后续只需输入名字自动关联
 - 任务列表：按日期分组显示，需反馈任务置顶且视觉突出
 - 日历视图：紧凑日历条，点击日期筛选任务
 - 任务操作：标记完成、打标（优先级/备注）
 - 常用Cooper：管理员设置的 + 用户自添加的链接
 - 反馈提醒：需反馈任务截止前2小时弹窗+震动提醒
 
 ### 管理端
 - 任务管理：新增/编辑/删除任务，支持公司筛选
 - 公司管理：增删公司、管理公司成员
 - 常用Cooper：添加全局或指定公司的Cooper链接
 - 审批管理：处理用户公司切换申请
 
 ### 移动端
 - 功能与用户端完全一致，底部抽屉式交互
 - Tab切换：任务 / 常用Cooper / 日历

 ## 注意事项
 - 数据存储在浏览器 localStorage，清除浏览器数据会丢失所有记录
 - 同一浏览器不同标签页共享数据，实时更新
 - 如需多设备同步，需部署后端存储（当前版本不支持）
