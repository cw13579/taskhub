# TaskHub 渠道任务协作平台 -- 基准源码 V1.4

## 项目概述

TaskHub 是一个轻量级的渠道任务协作平台，用于渠道经理向合作伙伴下发司机外呼任务（Cooper 表格）并进行进度跟踪。管理员和用户分别为独立入口，数据通过浏览器 localStorage 本地存储。

**V1.4 新增**：管理后台任务列表增加**复制任务**功能，点击即可基于已有任务快速创建新任务。

## 运行环境与启动

### 依赖环境

- 任何现代浏览器（Chrome 90+ / Edge 90+ / Firefox 90+ / Safari 15+）
- 无后端服务依赖
- 无需安装任何运行时或依赖包

### 本地启动方式

1. 直接双击 `index.html`（伙伴端）或 `admin.html`（管理后台）
2. 或拖拽文件到浏览器窗口
3. 手机浏览器打开 `index.html` 自动跳转 `mobile.html`

### 访问参数

| 参数 | 用途 |
|------|------|
| `?desktop=1` | 手机端强制使用桌面版 |
| `?mobile=1` | 强制切回移动版 |

## 数据存储路径

| 存储位置 | 数据类型 | 用途 |
|---------|---------|------|
| `localStorage.th3_global` | JSON | 任务、公司、审批、Cooper 链接全局数据 |
| `localStorage.th3_users` | JSON | 用户个人数据（备注、自建 Cooper 等） |
| `localStorage.th3_current_user` | String | 当前登录用户名 |

## 核心功能

### 角色划分（独立入口）

- **管理后台** (`admin.html`): 日历视图、新增/编辑/删除/复制任务、公司管理、Cooper管理、审批管理、GitHub数据同步
- **伙伴端** (`index.html` / `mobile.html`): 查看本公司任务、标记完成、打标备注、添加个人Cooper链接

### 功能模块

1. **日历视图** -- 月历展示，有任务的日期显示绿点，点击日期可筛选
2. **任务管理** -- 创建/编辑/删除/复制任务（含公司、伙伴、Cooper 链接、反馈要求）
3. **公司管理** -- 维护公司列表和成员
4. **状态流转** -- 点击勾选框：待处理 <-> 已完成
5. **打标/备注** -- 修改优先级、添加备注标签
6. **常用 Cooper** -- 管理员统一添加 + 用户自添加
7. **GitHub 数据同步** -- 侧栏面板手动 Push/Pull 数据到 GitHub 仓库
8. **移动端适配** -- 手机浏览器自动跳转移动版
